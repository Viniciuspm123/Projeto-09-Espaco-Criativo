# Projeto 09 – Espaço Criativo

## Descrição
Este projeto é um **site de portfólio/empresa criativa**, que apresenta a galeria de trabalhos, depoimentos de clientes e um formulário de contato.  
O objetivo é criar um layout moderno e responsivo, fácil de navegar e visualmente agradável.

---

## Funcionalidades
- **Home**: Apresenta uma mensagem de boas-vindas e chamada para contato.
- **Galeria**: Mostra imagens do portfólio ou projetos criativos.
- **Depoimentos**: Exibe feedbacks de clientes com cards visuais.
- **Contato**: Formulário para que o usuário envie seu nome e e-mail.
- **Footer**: Informações de direitos autorais.

---

## Tecnologias utilizadas
- HTML5 semântico
- CSS3 (arquivo externo `style.css`)
- Google Fonts (Inter)
- JavaScript (para funcionalidades futuras e formulários)
